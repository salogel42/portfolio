Useful Keyboard shortcuts:
  • Command + Tab to switch from one program to another!
  • Command + R to refresh (in most browsers)
  • Command + S to save (in most programs, including atom)

    • or to avoid saving all the time, hit Command + , in Atom, click "Packages", then type AutoSave. Click on the result, then check the checkmark next to enable

(On Windows, use control instead of command for these)
